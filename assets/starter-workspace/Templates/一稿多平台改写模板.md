# 一稿多平台改写模板

这个模板用于把一个已经有真实素材的核心选题，分别改写成小红书、抖音、公众号、视频号和知乎版本。

一稿多平台不是复制粘贴，也不是简单缩短。需要保留同一个事实和立场，再根据平台重新组织标题、开头、结构、语言和结尾。

## 一、先整理核心素材

```text
核心选题：
一句话观点：
目标人群：
读者当前问题：
真实经历或案例：
关键步骤或方法：
事实与数据：
希望读者采取的动作：
必须保留：
明确不要：
```

## 二、先建立“内容母稿”

母稿只需要回答：

1. 发生了什么；
2. 为什么值得讲；
3. 最重要的判断是什么；
4. 有哪些真实依据；
5. 读者下一步能做什么。

母稿不是任何平台的最终稿，它负责统一事实、观点和素材边界。

## 三、五个平台的改写重点

| 平台 | 开头重点 | 主体重点 | 结尾重点 |
| --- | --- | --- | --- |
| 小红书 | 场景、痛点、获得感 | 清单、步骤、个人体验 | 收藏价值、自然互动 |
| 抖音 | 前3秒人群/冲突/结果 | 短句、节奏、画面变化 | 一个明确动作 |
| 公众号 | 经历、问题、完整叙事 | 深度解释、方法、案例 | 总结判断或资料承接 |
| 视频号 | 真人身份、真实问题 | 信任、经验、清楚观点 | 关系建立和下一步 |
| 知乎 | 直接回答问题 | 论证、案例、边界 | 回到原问题 |

## 四、可直接复制的提示词

```text
请把下面的核心素材分别改写成小红书、抖音、公众号、视频号和知乎版本。

核心选题：[填写]
一句话观点：[填写]
目标人群：[填写]
读者当前问题：[填写]
真实经历或案例：[填写]
关键步骤或方法：[填写]
事实与数据：[填写]
希望读者采取的动作：[填写]
必须保留：[填写]
明确不要：[填写]

请先读取 AGENTS.md、About_me 和五个平台模板。

执行要求：
1. 先整理一份内容母稿，确认事实、观点和素材边界；
2. 五个平台分别重新组织，不要只做长短变化；
3. 小红书输出标题、封面、正文、话题和配图建议；
4. 抖音输出钩子、口播、字幕、画面和发布文案；
5. 公众号输出标题、摘要、完整文章和文末承接；
6. 视频号输出标题、封面、口播、字幕和拍摄建议；
7. 知乎输出直接回答、完整论证和适用边界；
8. 五个版本分别保存到对应的 Outputs 文件夹；
9. 完成后说明五个平台的改写差异，并进行发布前检查。
```

## 五、统一性检查

- [ ] 五个平台的核心事实一致；
- [ ] 五个平台的核心观点没有互相冲突；
- [ ] 没有为了适配平台而编造新的经历；
- [ ] 每个平台都有独立的标题和开头；
- [ ] 抖音和视频号不是完全相同的脚本；
- [ ] 公众号和知乎不是简单复制；
- [ ] 行动引导符合各平台的内容关系；
- [ ] 五个文件已经分别保存。
