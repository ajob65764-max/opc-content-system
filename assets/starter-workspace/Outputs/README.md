# Outputs 使用规则

这里保存已经形成的正式交付物，而不是随手记录和零散灵感。

## 文件夹

- `小红书/`
- `抖音/`
- `公众号/`
- `视频号/`
- `知乎/`

## 命名格式

```text
日期-平台-选题名称-版本.md
```

例如：

```text
2026-07-09-公众号-为什么要搭建自己的AI工作系统-v1.md
```

## 文件顶部建议记录

```text
状态：草稿/待审核/已发布
平台：
选题：
目标读者：
发布日期：
数据记录：
复盘结论：
```

同一个选题的不同平台版本分别保存。已经发布的文件不要直接覆盖，重大修改时新建版本。
